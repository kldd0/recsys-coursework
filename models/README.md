# Модели

Эта директория предназначена для хранения ML моделей и их оберток.

## Структура

```
models/
├── __init__.py
├── README.md
├── clothing_model.py          # Класс-обертка для вашей модели
├── clothing_embedding_model.pth  # Веса модели (добавьте ваш файл)
└── requirements_model.txt     # Дополнительные зависимости (опционально)
```

## Инструкция

1. Создайте класс-обертку в `clothing_model.py` (см. `docs/MODEL_INTEGRATION.md`)
2. Сохраните веса модели в эту директорию
3. Обновите `src/cache.py` для загрузки вашей модели
4. Запустите `scripts/generate_embeddings.py` для создания эмбеддингов

## Примеры

См. файл `docs/MODEL_INTEGRATION.md` для подробных примеров интеграции различных типов моделей.
